# MyBatis - Spring、Spring事务

# Spring整合MyBatis

  1. 添加依赖
  2. 在配置类中添加DataSource组件
  3. 在配置类中添加SqlSessionFactoryBean组件
  4. 在配置类中添加MapperScannerConfigurer组件

详细步骤：

  1. 添加依赖

    ① Druid数据库连接池

    <dependency>
    <groupId>com.alibaba</groupId>
    <artifactId>druid</artifactId>
    <version>1.2.11</version>
</dependency>

    ② mybatis-spring

    <dependency>
    <groupId>org.mybatis</groupId>
    <artifactId>mybatis-spring</artifactId>
    <version>2.1.1</version>
</dependency>

    ③ spring-jdbc

    <dependency>
    <groupId>org.springframework</groupId>
    <artifactId>spring-jdbc</artifactId>
    <version>${spring.version}</version>
</dependency>

    ④ 其他(spring-context、spring-test、aspectjweaver、mybatis、mysql-connector、junit)

  2. 在配置类中添加DataSource组件

    创建数据源连接池对象dataSource，对该对象设置数据库连接相关参数，然后返回该对象。

        @Bean
    public DataSource dataSource() {
        DruidDataSource druidDataSource = new DruidDataSource();
        druidDataSource.setDriverClassName("com.mysql.jdbc.Driver");
        druidDataSource.setUrl("jdbc:mysql://localhost:3306/test?useUnicode=true&characterEncoding=utf-8");
        druidDataSource.setUsername("root");
        druidDataSource.setPassword("root");
        return druidDataSource;
    }

  3. 在配置类中添加SqlSessionFactoryBean组件

    创建SqlSessionFactoryBean对象，并设置DataSource等属性，然后返回该对象。

        @Bean
    public SqlSessionFactoryBean sqlSessionFactoryBean(DataSource dataSource) {
        SqlSessionFactoryBean sqlSessionFactoryBean = new SqlSessionFactoryBean();
        sqlSessionFactoryBean.setTypeHandlersPackage("com.demo.spring.typehandler");
        sqlSessionFactoryBean.setDataSource(dataSource);
        return sqlSessionFactoryBean;
    }

  4. 在配置类中添加MapperScannerConfigurer组件

    创建MapperScannerConfigurer对象，并设置basePackage(扫描目录)、SqlSessionFactoryBeanName等属性，然后返回该对象。

        @Bean
    public MapperScannerConfigurer mapperScannerConfigurer() {
        MapperScannerConfigurer mapperScannerConfigurer = new MapperScannerConfigurer();
        mapperScannerConfigurer.setBasePackage("com.demo.spring.mapper");
        mapperScannerConfigurer.setSqlSessionFactoryBeanName("sqlSessionFactoryBean");
        return mapperScannerConfigurer;
    }
