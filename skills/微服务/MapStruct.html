# MapStruct

# 引入

对于前端一次数据请求，每层返回的用于封装数据的对象都不同。

控制层返回的对象被称为VO对象

业务层返回的对象被称为DTO对象

持久层返回的对象被称为PO对象

对象之间的转换可以使用MapStruct框架替我们完成转换，

MapStruct主要做的是PO到DTO的转换

从DTO到VO的转换一般是将DTO作为一个属性封装到一个通用的返回值对象

# 使用

① 引入依赖

<!--mapstruct-->
<dependency>
    <groupId>org.mapstruct</groupId>
    <artifactId>mapstruct-jdk8</artifactId>
    <version>1.3.0.Final</version>
</dependency>
<dependency>
    <groupId>org.mapstruct</groupId>
    <artifactId>mapstruct-processor</artifactId>
    <version>1.3.0.Final</version>
</dependency>

② 定义转化器接口

③ 使用定义的转化器接口

# 简单对象的转化

PO和DTO的属性个数和名称都相同

① 任务：User转化成UserDTO

@Data
public class User {
    String name;
    String address;
}

@Data
public class UserDTO {
    String name;
    String address;
}

② 定义转化器接口UserConverter

@Mapper(componentModel = "spring")
public interface UserConverter {
    //目标类 方法名      待转化类
    UserDTO userPO2DTO(User user);
}

③ 使用转化器接口

    @Autowired
    UserConverter userConverter;

    @Test
    public void simpleTest() {
        //构造数据，假装是从数据库中获取的
        User user = new User();
        user.setName("张三");
        user.setAddress("二仙桥");

        //使用转化器接口
        UserDTO userDTO = userConverter.userPO2DTO(user);
        System.out.println(userDTO);
    }

# 不同属性名的映射

PO和DTO的相对应的属性名不同

① 任务：Student转化成StudentDTO

@Data
public class Student {
    String name;
    Integer age;
    String school;
}

@Data
public class StudentDTO {
    String stuName;
    Integer stuAge;
    String school;
}

② 定义转化器接口StudentConverter

@Mapper(componentModel = "spring")
public interface StudentConverter {
    //source: PO中的属性名 target: DTO中的属性名
    @Mapping(source = "name", target = "stuName")
    @Mapping(source = "age", target = "stuAge")
    StudentDTO studentPO2DTO(Student student);
}

③ 使用转化器接口

    @Autowired
    StudentConverter studentConverter;

    @Test
    public void fieldMapTest() {
        //构造数据，假装是从数据库中获取的
        Student student = new Student();
        student.setName("李四");
        student.setAge(18);
        student.setSchool("武汉大学");

        //使用转化器接口
        StudentDTO studentDTO = studentConverter.studentPO2DTO(student);
        System.out.println(studentDTO);
    }

# 多个不同类型源对象的转化

多个PO对象转化成一个DTO对象（多个PO对象给一个DTO对象提供数据）

① 任务：Student、User转化成StudentMember

@Data
public class StudentMemberDTO {
    String name;
    Integer age;
    String school;
    String address;
}

② 定义转化器接口

@Mapper(componentModel = "spring")
public interface StudentMemberConverter {

    //数据来自多个对象的话，使用对象名.属性名指定source
    @Mapping(source = "student.name", target = "name")
    @Mapping(source = "student.age", target = "age")
    @Mapping(source = "student.school", target = "school")
    @Mapping(source = "user.address", target = "address")
    StudentMemberDTO studentPO2DTO(Student student, User user);
}

③ 使用转化器接口

    @Autowired
    StudentMemberConverter studentMemberConverter;

    @Test
    public void MultiObjTest() {
        //构造数据，假装是从数据库中获取的
        Student student = new Student();
        student.setName("李四");
        student.setAge(18);
        student.setSchool("武汉大学");

        User user = new User();
        user.setName("李四");
        user.setAddress("火星");

        //使用转化器接口
        StudentMemberDTO studentMemberDTO = studentMemberConverter.studentPO2DTO(student, user);
        System.out.println(studentMemberDTO);
    }

# 转化复杂对象

类似于深度克隆的深度转化，一个DTO对象的成员变量也是DTO对象的情况

① 任务：Student、Course转化成StudentCourseInfoDTO

@Data
public class Course {
    String name;
    Integer hour;
}

@Data
public class CourseDTO {
    String name;
    Integer hour;
}

@Data
public class Student {
    String name;
    Integer age;
    String school;
}

@Data
public class StudentCourseInfoDTO {
    String name;
    Integer age;
    String school;
    Course course;
}

② 定义转化器接口StudentCourseInfoConverter

@Mapper(componentModel = "spring")
public interface StudentCourseInfoConverter {

    @Mapping(source = "student.name", target = "name")
    @Mapping(source = "student.age", target = "age")
    @Mapping(source = "student.school", target = "school")
    StudentCourseInfoDTO studentCoursePO2DTO(Student student, Course course);
}

当存在成员变量需要转化时，转化器会自动调用已经定义好的转化器接口方法。

@Mapper(componentModel = "spring")
public interface CourseConverter {

    CourseDTO coursePO2DTO(Course course);
}

③ 使用转化器接口

@Autowired
StudentCourseInfoConverter studentCourseInfoConverter;

@Test
public void ObjectFieldTest() {
    //构造数据，假装是从数据库中获取的
    Student student = new Student();
    student.setName("李四");
    student.setAge(18);
    student.setSchool("武汉大学");

    Course course = new Course();
    course.setName("Java");
    course.setHour(70);
	
    //使用转化器接口
    StudentCourseInfoDTO studentCourseInfoDTO = studentCourseInfoConverter.studentCoursePO2DTO(student, course);
    System.out.println(studentCourseInfoDTO);
}

输出：

StudentCourseInfoDTO(name=李四, age=18, school=武汉大学, course=Course(name=Java, hour=70))

# 转化List

例如：List转换成List

① 任务：List转换成List

@Data
public class Course {
    String name;
    Integer hour;
}

@Data
public class CourseDTO {
    String name;
    Integer hour;
}

② 定义转化器接口

@Mapper(componentModel = "spring")
public interface CourseConverter {

    CourseDTO coursePO2DTO(Course course);

    List<CourseDTO> coursePOs2DTOs(List<Course> courses);
}

③ 使用转化器接口

    @Autowired
    CourseConverter courseConverter;

    @Test
    public void listMapTest() {
        //构造数据，假装是从数据库中获取的
        Course firstCourse = new Course();
        firstCourse.setName("C++");
        firstCourse.setHour(120);
        Course secondCourse = new Course();
        secondCourse.setName("python");
        secondCourse.setHour(80);
        List<Course> courses = Arrays.asList(firstCourse, secondCourse);
        
        //使用转化器接口
        List<CourseDTO> courseDTOS = courseConverter.coursePOs2DTOs(courses);
        System.out.println(courseDTOS);
    }

运行结果

[CourseDTO(name=C++, hour=120), CourseDTO(name=python, hour=80)]
