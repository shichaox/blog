# 定义切面

  1. 通过ProceedingJoinPoint对象获取Signature，然后通过Signatrue获取method对象，获取该方法的注解，然后根据注解中的prefix值构造redis的key
  2. 获取key值对应的Bucket代理对象，然后获取对应的redis的value
  3. 如果value不为空，就返回该value值
  4. 如果value值为空，就准备从数据库中查找
  5. 通过redissonClient.getLock获取锁
  6. 加锁，双重检查，重复3中的判断(放到try catch代码块中)
  7. 执行method方法(通过ProceedingJointPoint的proceed方法)
  8. 判断查到的数据是否为null，为null就发生了缓存穿透，返回空的对象
  9. 不为null就将数据库中查到的数据放到redis中去
  10. finally代码块中释放锁
  11. 返回数据库查到的数据对应的对象

# 布隆过滤器

使用

// 根据指定名称(key)获取布隆过滤器
RBloomFilter rbloomFilter = redissonClient.getBloomFilter("xxx");
// 初始化布隆过滤器，预计统计元素数量为100000，期望误判率为0.01
rbloomFilter.tryInit(100000, 0.01);
// 向布隆过滤器中添加目标元素
rbloomFilter.add(目标元素);
// 判断目标元素是否存在，返回false表示不存在
boolen exists = rbloomFilter.contains(目标元素);
