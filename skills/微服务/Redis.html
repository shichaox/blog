# Redis

Redis: 是一个NoSQL数据库、纯内存数据库，存储key-value键值对。

NoSQL: not only SQL(非关系数据库)

Redis是纯内存数据库，数据存储在内存中，并不意味着关机后它的数据就失效了，它也有持久化策略，可以按照某种策略将数据持久化(存储)到磁盘。

# string

#set/get设置、获取
set key value
get key

#mset/mget批量设置、获取键值对
mset key1 value1 key2 value2
mget key1 key2

#incr/incrby自增、额外设置自增的步长
incr key
incr key increment

#decr/decrby自减、额外设置自减的步长
decr key
decr key decrement

#setex/setnx设置过期时间、当key不存在的时候就设置值，不会覆盖原来的值
setex key seconds value
setnx key value

# list

本质是一个双向链表，list支持从两端推入或弹出元素

#lpush/lpop
rpush/rpop

#llen查看链表的元素数量
llen key

#查看某一个范围内的元素
lrange key start stop

#获取指定下标的元素
lindex key index

#在指定元素前或后插入一个元素
linsert key before|after pivot value

#从左端推入，当list存在时才生效
lpushx key value

#修改指定位置的元素的值
lset key index value

#删除前几个指定的元素
lrem key count val

# Redisson

① 引入依赖

<dependency>
	<groupId>org.redisson</groupId>
	<artifactId>redisson</artifactId>
	<version>3.11.1</version>
</dependency>

② 使用

通过Redisson操作redis数据库中的数据(key-value)，首先要获取要操作数据的代理对象，然后对该对象进行操作，实现对数据库中的数据的操作。也就是说，对这些代理对象添加、获取数据就等于对redis数据库进行数据的添加和获取。

list、set、hash通过RedissonClient获取到的代理对象分别可以对应List、Set、Map数据结构，对他们的操作方法也和List、Set、Map一样。

public class RedissonTest {

    RedissonClient redissonClient;

    @Before
    public void before() {
        Config config = new Config();
        SingleServerConfig serverConfig = config.useSingleServer()
                .setAddress("redis://localhost:6379");
        config.setCodec(new JsonJacksonCodec());
        redissonClient = Redisson.create(config);
    }

    //测试操作string类型
    @Test
    public void testString() {
        //获取string类型的代理对象，代理的键值对的key为test
        RBucket<String> test = redissonClient.getBucket("test");
        test.set("hello world");
    }

    //测试操作list类型
    @Test
    public void testList() {
        //获取list类型的代理对象，代理的键值对的key为user
        List<String> userList = redissonClient.getList("user");
        userList.add("张三");
        userList.add("李四");
    }

    //测试操作set类型
    @Test
    public void testSet() {
        //获取set类型的代理对象，代理的键值对的key为season
        Set<Object> seasonSet = redissonClient.getSet("season");
        seasonSet.add("春");
        seasonSet.add("夏");
        seasonSet.add("秋");
        seasonSet.add("冬");

        //seasonSet.remove("冬");
    }

    //测试操作hash类型
    @Test
    public void testHash() {
        //获取hash类型的代理对象，代理的键值对的key为token
        RMap<String, Object> tokenMap = redissonClient.getMap("token");
        tokenMap.put("张三", 666);
        tokenMap.put("李四", 123);
        tokenMap.put("王五", 789);
    }

    //测试操作Zset(sortedset)类型
    @Test
    public void testZset() {
        //获取Zset类型的代理对象，代理的键值对的key为rank
        RScoredSortedSet<Object> rankZset = redissonClient.getScoredSortedSet("rank");
        rankZset.add(60, "张三");
        rankZset.add(90, "李四");
        rankZset.add(70, "王五");
    }
}
