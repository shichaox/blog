# DOM

DOM：文档对象模型(Document Object Model)

DOM定义了访问和操作HTML文档的标准方法。DOM以树结构表达HTML文档。

当网页被加载时，浏览器会创建页面的DOM。DOM被构建为对象的树。在DOM中，HTML的每一个标签，文本，都会被创建为一个结点，根据包含关系来确定在DOM树中的父子关系。通过修改结点，可以修改HTML页面内容。

通过可编程的对象模型，JavaScript可以改变页面中的所有HTML元素、所有HTML属性、所有CSS样式，JavaScript能够对页面中所有的事件做出反应。

# 获取结点（查找HTML元素）

  1. 通过id查找

    document.getElementById("")

  2. 通过类名查找

    document.getElementByClassName("")

  3. 通过标签名查找

    document.getElementByTagName("")

# v-bind

单项绑定，可以把vue中的对象的属性绑定到一个html代码的属性上。

只有vue对象中的属性的变化会影响html的属性，而html代码中的属性变化不会使vue对象中的属性也发生变化

# v-model

双向绑定，可以把一个vue对象中的属性双向绑定到对应html代码的表单元素的value上，两者会互相影响，其中任一属性发生改变，另外一个也随之改变。

# v-on

用于监听一个事件，把这个事件触发vue对象中的methods属性中提供的方法
