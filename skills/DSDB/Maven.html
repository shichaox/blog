# Maven

Maven是什么？

Maven是一个项目管理工具，它可以

  1. 管理项目所需要的jar包

    maven在本地有一个仓库用于存放jar包，使用maven管理项目后，我们使用jar包就不用自己手动添加了，只需要在pom.xml文件中声明你需要的jar包（依赖），maven就会帮你自动下载和导入，已经下载过的jar包不需要重复下载。

    导入依赖格式：

    	<dependencies>
        <dependency>
            <groupId>mysql</groupId>
            <artifactId>mysql-connector-java</artifactId>
            <version>8.0.30</version>
        </dependency>
    </dependencies>

  2. 将项目打包

    将依赖的jar包放到指定的目录，并按指定格式将项目打包。

使用maven管理依赖的好处：

  1. 无需手动下载添加依赖
  2. 避免不同项目重复导入相同的jar包
  3. 在更换项目编译环境时，maven可以自动按配置导入jar包

将当前项目改为Maven项目

  1. 在项目下创建一个pom.xml文件，对该文件右键Add as Maven Project
  2. 取消src的source标记
  3. 在src下面创建一个main、test目录
  4. 在main目录下创建java目录(标记为Sources root)、resources目录(标记为Resources root)
  5. 在test目录下创建java目录(标记为Test Sources root)、resources目录(标记为Test resources root)
  6. 把原来src目录下的源文件移动到main/java目录下
